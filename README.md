# Telescope

A React-based project built with Vite and Storybook. This project is designed for streamlined development with tools such as TailwindCSS, React Query, Jotai, and more.

## Getting Started

### Prerequisites

Ensure the following are installed on your system:

- **Node.js**: ^16.0.0 or higher
- **npm** (comes with Node.js), or use **pnpm** or **yarn** if preferred.

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd telescope
   ```
1. Clone the repository:
   ```bash
   npm install
   ```

## Scripts

The following scripts are available in the `package.json` file:

### Start Development Server

Runs the Vite development server on `http://localhost:5173` (default):

```bash
npm run dev
```

### Start Storybook

Launches Storybook for component development and testing on `http://localhost:6006`:

```bash
npm run storybook
```
